🔍 Cloud native security란?

클라우드 네이티브 보안은 클라우드 네이티브 애플리케이션(Cloud Native Application)의 설계, 개발, 배포, 실행 환경에서 발생하는 보안 문제를 관리하고 보호하는 체계와 기술입니다.

📍 배경 및 필요성

클라우드 네이티브 아키텍처의 확산 : 클라우드 네이티브는 유연한 확장성을 가지며 자동화된 리소스 관리가 가능하여, IT 서비스 운영 효율성 크게 향상했습니다.

기존 보안 접근 방식의 한계 : 기존의 보안 체계는 정적이고 경계 기반(perimeter-based) 보안 모델에 의존하여, 클라우드 환경의 동적이고 분산된 특성을 효과적으로 다루지 못한다는 단점이 있습니다.
